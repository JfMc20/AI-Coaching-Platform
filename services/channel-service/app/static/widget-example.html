Welcome to the AI Coaching Platform Widget

This is a demonstration of our embeddable AI chat widget. The widget allows your website visitors to interact with your AI coaching assistant directly on your site.

  • Real-time WebSocket communication
  • Responsive design for all devices
  • Customizable themes and colors
  • Easy integration with any website
  • Persistent conversation history
  • Connection status indicator
  • Typing indicators
  • Mobile-friendly interface

How to Integrate

Add the following code to your website to embed the AI chat widget:

<!-- AI Chat Widget Configuration --> <script> window.AIChatWidgetConfig = { apiUrl: 'ws://localhost:8004', widgetId: 'your-widget-id', position: 'bottom-right', // 'bottom-right' or 'bottom-left' theme: 'light', // 'light' or 'dark' primaryColor: '#007bff', welcomeMessage: 'Hello! How can I help you today?', placeholder: 'Type your message...' }; </script> <!-- Load the Widget --> <script src="https://your-domain.com/widget.js"></script>

Configuration Options

  • apiUrl - WebSocket URL for your AI service
  • widgetId - Unique identifier for your widget
  • position - Widget position on screen
  • theme - Visual theme (light/dark)
  • primaryColor - Main color for the widget
  • welcomeMessage - Initial greeting message
  • placeholder - Input field placeholder text
